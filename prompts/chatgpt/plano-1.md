# Guia Estratégico de Recolocação Tech 2026 - ChatGPT (Plano 1)

## Objetivo: Criar uma máquina de aquisição de oportunidades

### 1. Estratégia de Bifurcação dos Perfis
Em 2026, perfis generalistas são penalizados por ATSs. A estratégia é criar duas personas independentes.

#### Persona A — Desenvolvedor Fullstack
- **Posicionamento:** Fullstack com 3+ anos em SaaS e aplicações modernas.
- **Keywords:** React, Next.js, TypeScript, Node.js, PostgreSQL, Docker, CI/CD, Turborepo, Supabase, Electron, SaaS, Tailwind.
- **Experiência Cybernetrs:** Minimizar o suporte. Colocar como "Experiência em operações de TI e monitoramento de ambientes críticos".

#### Persona B — Técnico de TI / Infraestrutura
- **Posicionamento:** Suporte N2, redes, automação e monitoramento.
- **Keywords:** Active Directory, TCP/IP, VPN, Windows Server, Linux, Zabbix, GLPI, PowerShell, Python.
- **Experiência Freelance:** Descrever como desenvolvimento de scripts de automação e ferramentas internas.

### 2. Portfólio e Presença Online
- **Domínio Principal:** `dionatha.com.br` exclusivo para DEV.
- **Subdomínio:** `infra.dionatha.com.br` para TI/Suporte.
- **LinkedIn:** Perfil híbrido inteligente. Título: "Desenvolvedor Fullstack | Automação | Infraestrutura e Operações de TI".
- **GitHub:** Foco total em DEV. Criar repo `automacao-ti` para conectar os mundos.

### 3. Ecossistema de Candidaturas
- **Sites DEV:** LinkedIn, Gupy, Revelo, GeekHunter, Programathor.
- **Sites TI:** Catho, Indeed, InfoJobs, Stefanini, TIVIT.
- **Automação (Máquina de Emprego):**
    - Crawler em Python (Playwright/BeautifulSoup).
    - Score ATS via GPT.
    - CRM pessoal em Next.js + PostgreSQL para controle de funil.

### 4. Estratégia "80/20" Semanal
- **Seg/Ter/Qui:** Candidaturas (10 DEV, 5 TI) + Networking.
- **Qua:** Atualização de GitHub e LinkedIn.
- **Sex:** Follow-up.
- **Sábado:** Novo projeto ou estudo.
