# Melhor Versão - Claude (Consolidado)

Baseado no Plano 2 (único disponível), este plano foca em **Autoridade Técnica** e **Narrativa de Especialista Dual**.

## 1. O Diferencial: "Full-System Engineer"
A estratégia da Claude é posicionar Dionatha não apenas como um Dev, mas como alguém que domina o ciclo de vida completo do software, da rede ao pixel.

### Estratégia de Portfólio
- **O Toggle:** Uma experiência de usuário que permite alternar entre "Engenheiro de Software" e "Especialista em Infraestrutura". Isso demonstra versatilidade sem parecer amador.
- **Destaque:** Projetos como o sincronizador do TinyERP (Detcheler) servem de ponte entre os dois mundos.

## 2. Automação de Alto Nível
- **JSON Resume:** Uso centralizado de dados estruturados para permitir que IAs (como o Claude) reescrevam o currículo cirurgicamente para cada vaga.
- **Anti-Bot Gupy:** Implementação de técnicas de simulação humana com Playwright (delays randômicos e movimentos de mouse) para garantir que as candidaturas automáticas não sejam descartadas.

## 3. Branding de Especialista (Blog)
A Claude sugere uma série de artigos que resolvem "dores de cabeça" de Tech Leads:
- Sincronização de dados sem SDK oficial.
- Escolha de banco de dados baseada em casos reais.
- Uso de Electron para automação industrial/desktop.

## 4. Cold Outreach Cirúrgico
Foco em identificar a stack da empresa (Tiny ERP, etc) e oferecer uma solução baseada em experiências passadas, transformando o "contato frio" em uma "proposta de valor".
