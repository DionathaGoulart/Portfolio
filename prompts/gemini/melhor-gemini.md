# Melhor Versão - Gemini (Consolidado)

Esta versão foca na **Segurança Algorítmica** e na **Eficiência de Conversão**.

## 1. A Estratégia de "Invisibilidade Dual"
O diferencial do Gemini é o cuidado para não ser detectado como "robô" ou "generalista" pelos sistemas:
- **Separação de Currículos:** PDF A (Dev) limpo e focado; PDF B (Infra) técnico e focado.
- **Extensão de Browser Personalizada:** Em vez de bots de spam, uma extensão que auxilia o preenchimento manual simulando comportamentos humanos (mouse e digitação).

## 2. Portfólio como Hub de Valor
- **Split Hero:** A entrada do site permite que o visitante escolha o "canal" de interesse.
- **Projetos de Integração:** O projeto **Detcheler** é usado como a peça central que valida ambas as habilidades (Dev e Infra).

## 3. Automação de Match Score
- Uso de LLM local ou API para analisar a Job Description e ajustar automaticamente os *bullets* do currículo, garantindo que as experiências batam com o que o recrutador busca (Cluster Semântico).

## 4. Networking de Alta Intencionalidade
- Cold emails que não pedem emprego, mas oferecem "5 minutos de papo sobre um desafio específico" (ex: Monorepos ou Automação de SLA).
